# FUNDAMENTOS_DE_PYTHON

Este repositorio contiene las prácticas, ejercicios y el proyecto final desarrollados durante el curso **"Fundamentos de Python"** impartido por **INFOTEC** (Agencia de Transformación Digital y Telecomunicaciones).

## Descripción del Curso
El curso tuvo una duración de **30 horas**, cubriendo desde las bases de la lógica de programación hasta la gestión de datos con librerías especializadas y Programación Orientada a Objetos.

## Estructura del Proyecto

El contenido está organizado por unidades, siguiendo el plan de estudios del curso:

### Unidad 1: Introducción a Python y Estructuras de Control
* **1A. Ejercicio automatizado:** Pruebas de sintaxis básica.
* **1B. Estructuras de control:** Implementación de condicionales y bucles, junto con el manejo de tipos de datos.

### Unidad 2: Funciones y Estructura de Datos
* **2A. Ejercicio de Funciones:** Modularización de código y uso de colecciones de datos.

### Unidad 3: Archivos y Manejo de Datos con Pandas
* **3A. Prueba de Pandas:** Manipulación de DataFrames y análisis de datos.
* **3B. Práctica de lectura:** Ejercicios de carga y lectura de archivos externos.

### Unidad 4: Programación Orientada a Objetos (POO)
* **4A. Calculadora básica:** Implementación de una calculadora utilizando clases, métodos y atributos.
* **4B. Proyecto Final:** Desarrollo de un script integral en Python que aplica los conocimientos adquiridos en el curso.

## Tecnologías utilizadas
* **Lenguaje:** Python 3.11.7
* **Librerías:** Pandas
* **Paradigma:** Programación Orientada a Objetos (POO)
